Usage

    Create Map (add points and connections or use Rapid Deployment )
    Mark start and end points
    Choose alg
    Click Path -> Find


Colors

    Green - start
    Red - end
    Yellow - path chosen
    Bleu - point analyzed
    Pink - Front - next possible hop
